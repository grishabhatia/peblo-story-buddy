# 🚀 Peblo Story Buddy

AI Story Buddy & Quiz Component for Peblo - built with Flutter.

---

## ✨ Features

- 📖 Text-to-Speech story narration
- ❓ Data-driven quiz (supports 3-5 options)
- 🎊 Confetti celebration on correct answer
- 🔄 Shake animation on wrong answer
- 🎨 Kid-friendly UI with Buddy avatar

---

## 🛠️ Tech Stack

- Flutter + Provider
- flutter_tts
- confetti

---

## 🚀 Run

```bash
flutter pub get
flutter run
